자동차 세큐리티 프로그램

자동차 클래스
	String name;
	int pw;
	boolean door; // 문이 열려있는지 확인
	boolean engine; // 시동이 켜져있는지 확인
	
	문 열기
	opendoor() 메소드
		문이 잠겨있다면, 열어준다.
		문이 열려있다면, 잠궈준다.
		
		- 메인에서 만들기
		단, 문이 잠겨있다면 비밀번호를 확인한 후 열어줘야한다.
		비밀번호를 입력할 기회는 총 3번
		3번 이상되면 경찰 출동 출력하면서 프로그램 종료

	engineStart() 메소드
		문이 열려있고, 시동이 꺼져있는 상태에서만 시동이 켜진다. <- 출력
		engine > true로 변경

	engineStop() 메소드
		문이 열려있고, 시동이 켜져있는 상태에서만 시동 켜짐 <- 출력
