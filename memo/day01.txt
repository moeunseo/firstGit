JAVA
> JVM (Java Virtual Machine): JAVA로 만든 프로그램 실행
> JRE (Java Runtime Environment): JVM포함, JVM 실행할 때 필요한 라이브러리, 파일 등을 소유
> JDK (Java Development Kit): JRE포함, JRE 외에 개발에 필요한 도구들을 가지고 있다. (컴파일러 등)

JAVA 설치하기
구굴 > JDK17 설치 > 바로 위에 나오는 링크 > 17.0.10 다운로드(window installer)
> 다운로드 파일 잘라서(crtl+x) resource로 복사

JAVA 버전 확인하기
cmd > java -version > 설치한 자바 버전 뜨면 성공

환경변수는 디폴트로 잡힘 > 제일 최신 버전으로

eclipse=IDE (Integrated Development Environment: 통합개발환경)
> 편하게 개발하기 위해 사용
이클립스 - 무료버전 (정부 공인 프로젝트에서 사용)
IntellJ - 유료버전

이클립스 다운로드는 구글에 쳐봐~
구글 > eclipse 검색 후 최상단 링크 접속 > 화면 중앙 왼쪽 Download packages
> 오른쪽 아래 2023-03 > Eclipse IDE for Enterprise Java and Web Developers

**상대경로(바로가기)랑 절대경로 의미 알고 있기

관점 변경 
JAVA EE > JAVA 로 변경

인코딩 변경 UTF-8 디폴트로 설정
window > preference > encoding 검색 > Workspace > 왼쪽하단 인코딩 방식 변경

프로젝트의 기븐 구조
프로젝트 > 패키지(소문자 시작) > 클래스(대문자 시작) > 메소드 > 소스코드


프로젝트 import하기
압축파일 다운 > 왼쪽 위 file탭 > import > General > Existing Project into Workspace
> Select archie File > Browse > 다운로드한 압축 파일 선택 > 이름은 같지 않게 import하기
