이클립스 단축키 정리
프로젝트 폴더 생성: alt+shift+n > java project > 이름 설정, 모듈 체크 해제 > finish

패키지 생성: crtl+n > package 검색 후 엔터 > 이름(소문자) 설정 > 엔터

클래스 생성: crtl+n > class 선택 > 이름(대문자) 설정 > 엔터

메인 메소드 생성: 클래스 영역 안에서 main작성 후 crtl+space

출력 메소드 생성: syso > crtl+space

전체 프로젝트 저장: crtl+shift+s
실행: crtl+F11

메인 메소드: 자바 프로그램 실행 시 메인 메소드 안에 있는 영역이 실행
	   중괄호로 영역 구분 세미콜론을 찍어 문장이 끝났음을 알려줌

자료형(data type): 값의 종류

	타입	용량(byte)	예시
정수형	int	4		10, 0, -10, 55, 321 ...
	long	8		10L, -123L
실수형	float	4		3.14F, 0.0F
	double	8		3.14, 0.0
논리형	boolean	1		true, false
문자형 	char	1		'a', 'A', '0', 'b', 'c' ...
문자열	String	???		"이창섭", "모은서", "123", "a" ...
> 문자열은 클래스임 why? 대문자로 시작했어 man~

변수 (Variable): 값을 저장하는 저장 공간
> 오직 하나의 값만 저장 가능
> 변수를 만들 때 설정한 자료형만 저장 가능
> 변수는 영역 상단에 선언!

변수: 변하는 숫자
상수: 변하지 않는 숫자 > 대문자로 선언

**형변환 (기본 자료형들에 해당)
	자동 형변환: 연산의 결과를 이용하여 형변환
		> 정수 + 정수 = 정수
		> 정수 +  실수= 실수

	강제 형변환: 직접 변환하고 싶은 자료형을 명시
		(자료형) 값
		(double) 49
